Static Site with Pelican

Date Wed 11 February 2015 Tags Pelican / Static-Site

I had been searching for a good static site generator for almost a year now. I tried Jekyll first, well it was fine, generated cool website, but the problem was it's initial installation and configuration. Gems, gems and gems all the time, too many gems, too complex configurations (at least for me). So I discontinued using it, even though it was nice. :(

Finally, after waiting, searching, reading and trying to overcome the laziness, I found Pelican. A static site generator made using python which utilizes jinja2 templating. Had to play around and configure for nearly a day before I was able to get satisfied with the stuffs like theme, appearance, structure etc.

I have used pelican-bootstrap3 theme for this website with bootstrap-simplex (theme inside theme?). Modified bootstrap-simplex a little to match my liking. And the site is hosted in github, gh-pages.
